    zmiennej j jako str
    zmiennej k jako float
    zmiennej l jako bool

    W pliku ex4.py wczytaj ze napis ze standardowego wejścia. W kolejnych liniach wydrukuj:

    Długość napisu

    True, gdy napis można skonwertować do typu float, False w przypadku przeciwnym
